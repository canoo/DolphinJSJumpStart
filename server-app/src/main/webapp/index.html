Dolphin.js

Welcome to Dolphin.js

Jump Start project

This project exemplifies the interplay of a standard Dolphin server component and various Dolphin web client.

The Editor

The editor is the master client which allows adding and editing of temperature measuresments. To run the editor, point your browser at:

http://localhost:8080/dolphinServer/editor/

The Web Reader

The web reader is a client that displays temperature measuresments as they are added and changed. The web reader is a pure web client, but not optimized for mobile. To run the web reader, point your browser at:

http://localhost:8080/dolphinServer/reader/

The Mobile Reader

The mobile reader is a web client optimized for mobile touch UIs. To run the mobile reader, point your browser at:

http://localhost:8080/dolphinServer/mobile/
