siron-onboarding-mobile-app
===========================


Prerequisites
-------------

* a working node.js environment
* backbone 1.0.0
* handlebars 1.0.0-rc3
* requirejs 2.1.5
* underscore 1.4.4
* grunt 0.4.1

Installation
------------

* checkout
* run "grunt copy:dist" to copy the necessary files to the ios project
